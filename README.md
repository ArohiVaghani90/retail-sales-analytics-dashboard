# Retail Sales Analytics Dashboard

This project is an interactive retail sales analytics dashboard built using **Python** and **Streamlit**. It allows users to explore, analyze, and visualize retail sales data in an easy-to-understand way.

## Live Dashboard
You can view the live application here:  
https://retail-sales-analytics-dashboard-ubqefhhb4yyeafapnzbhyn.streamlit.app/

## Project Overview
The dashboard provides insights into retail sales performance, including:
- Sales trends over time  
- Category-wise and product-level analysis  
- Key metrics and visualizations to support data-driven decisions  

The goal of this project is to make retail data more accessible and easier to analyze through an interactive web interface.

## Technologies Used
- Python  
- Streamlit  
- Pandas  
- NumPy  
- SQLAlchemy  
- Matplotlib / Seaborn
